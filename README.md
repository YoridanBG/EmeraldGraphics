<h1 align="center">EmeraldGraphics</h1>

<h5 align="center">Support for non-Apple NVIDIA GPUs on OS X / macOS.</h5>
</br>

A [Lilu](https://github.com/acidanthera/Lilu) plug-in kernel extension for NVIDIA branded graphics processing units, and similar architectures found in OS X and macOS.

</br>
<h1 align="center">Purpose</h1>
</br>

Specifically designed to help debug, and patch various NVIDIA related kernel extensions present on the OS X and macOS to get off-the-shelf or non-natively supported graphics processing unit *variants* working on non-Apple hardware (NOT non-supported hardware). The intention is to support the earliest OS X versions from Tiger, to modern macOS.
